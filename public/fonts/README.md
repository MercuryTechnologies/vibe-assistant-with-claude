# Arcadia Font Family

This directory contains the Arcadia variable fonts used throughout the Mercury Vibe application.

## Files

- `ArcadiaText-VariableVF.woff2` - Variable font for body text (weights 100-900)
- `ArcadiaTextItalic-VariableVF.woff2` - Variable font for italic body text (weights 100-900)
- `ArcadiaDisplay-VariableVF.woff2` - Variable font for display/headings (weights 100-900)
- `ArcadiaDisplayItalic-VariableVF.woff2` - Variable font for italic display text (weights 100-900)

## Usage

The fonts are automatically loaded via `src/fonts.css` and are available through CSS variables:

- `--font-sans`: Arcadia Text (default for all body text)
- `--font-display`: Arcadia Display (for headings and display text)
- `--font-mono`: Arcadia Text (for monospace contexts)

## Variable Font Features

These are variable fonts supporting:
- **Weight axis**: 100 (Thin) to 900 (Black)
- **Style**: Regular and Italic variants
- **Format**: WOFF2 (optimized for web performance)
